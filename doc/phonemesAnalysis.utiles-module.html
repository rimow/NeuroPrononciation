getPhonemeDict(path)

source code 

:param path: chemin du dictionnaire (qui a une forme bien specifique, voir fichier data/classement). :return: un dictionnaire de la forme : {'phoneme1':[1,0], 'phoneme2':[0,1]...} La premiere valeur du tableau vaut 1 si le phoneme est une voyelle, 0 si c'est une consonne, 2 si c'est un silence La seconde valeur du tableau vaut 1 si le phoneme est voise, 0 si non voise, 2 si c'est un silence
